# Что такое understandable?

**Понимание кодовой базы как один Rust-бинарник.** Натрави на любой
исходный репозиторий — получишь навигируемый граф знаний, гайдед-тур,
семантический поиск и локальную веб-панель. Всё это отдаётся одним
исполняемым файлом ~40 МБ.

## Зачем он существует

`understandable` — это Rust-порт оригинального TypeScript-проекта
[`Understand-Anything`][orig]. TS-прототип доказал, что идея работает;
переписывание на Rust сделало её дешёвой в эксплуатации.

Что даёт порт:

- **Один бинарник, ноль рантайм-зависимостей.** Никаких
  `node_modules`, никакого Python venv, никакого LSP. React-UI панели
  встроен в бинарник на этапе компиляции через `rust-embed`.
- **Нативный tree-sitter.** Парсеры работают в одном процессе — без
  IPC, без перегона JSON между процессами. Tier-1-грамматики
  покрывают 11 языков сразу; ещё ~30 включаются через `--features
  all-langs`.
- **Производительность.** Репозиторий из 10k файлов разбирается за
  секунды. Эмбеддинги, поиск и обход графа работают in-process
  поверх in-memory IndraDB, упакованного в `tar.zst`-архив.
- **Детерминированный субстрат.** ID вершин выводятся через
  `Uuid::new_v5` от стабильных бизнес-ключей. Повторный запуск
  `analyze` на тех же входах даёт тот же граф.

## Что получаешь

После одного `understandable analyze` у тебя есть:

- **Типизированный граф знаний** — файлы, символы, модули, слои и их
  зависимости.
- **Семантический поисковый индекс** (если ещё запустить
  `understandable embed`) — ранжирует узлы по косинусной близости
  через OpenAI, Ollama или полностью офлайн ONNX-эмбеддинги.
- **Гайдед-тур** — эвристическое упорядочивание узлов, проводящее
  новичка по структуре проекта.
- **Интерактивная панель** на `http://127.0.0.1:5173` с графовым
  представлением, обозревателем соседей и поиском.
- **LLM-обогащение** — опциональные сводки по файлам, теги и оценки
  сложности через Anthropic, OpenAI-совместимые серверы или
  host-LLM твоей IDE (Claude Code, Cursor, Copilot, …).

## Быстрый старт

```bash
cargo install --git https://github.com/yaroher/understandable understandable \
  --features all-langs,local-embeddings

cd /path/to/your/repo
understandable init --preset cloud-full
understandable analyze
understandable dashboard
```

См. [Установка](./getting-started/install) для полного набора путей
установки и [Первый граф](./getting-started/first-graph) для
пошагового туториала.
